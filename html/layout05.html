레이아웃05

이번 레이아웃은 전체 영역이 들어간 구조입니다. 실제 사이트에서 이런 구조를 많이 사용하며, 컨테이너를 만들어서 가운데 영역을 설정합니다.

미디어쿼리를 이용한 반응형

클래스 container로 간략하게 코딩할 수 있다. 

* {
    margin: 0;
    padding: 0;
}
#header {
    height: 100px;
    background-color: #E0F2F1;
}
#nav {
    height: 300px;
    background-color: #80CBC4;
}
#section {
    height: 500px;
    background-color: #26A69A;
}
#footer {
    height: 100px;
    background-color: #00897B;
}
/* id/class 
    #/.
    한번만 쓸 수 있음 / 두개(중복) 이상 쓸 수 있음 
    전체적인 툴, 스크립트 --> id
    세부적인 부분, 반복적인 부분 --> class를 주로 씀 
    */

.container {
    width: 1200px;
    height: inherit;
    margin: 0 auto;
    background-color: rgba(0,0,0,0.3);
}
@media (max-width:1220px){
    .container{
        width: 96%;
    }
}
@media (max-width:768px){
    .container{
        width: 100%;
    }
}
