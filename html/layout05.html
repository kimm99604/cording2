레이아웃05

컨테이너를 이용한 float, flex, grid 레이아웃

float

원하는 곳에 float: left, right, none 등 을 준다.

float을 줄 땐 버그 방지를 위한 방법이 3가지 있다.
1. claer: both;
2. overflow: hidden;
3. .clearfix::before,
.clearfix::after {content: '' " ; display: block; line-height: 0;}
.clearfix::after {clear: both;}

float - 반응형

미디어쿼리를 통해 float 반응형을 만들 수 있다. 


@media (max-width:1220px){
    .container {
        width: 96%;

    .contents .cont1 {
        width: 30%;
        height: 780px;
        float: left;

flex

부모에 flex를 주면 자식들에게 적용된다.

flex - 반응형

미디어쿼리를 통해 flex 반응형을 만들 수 있다.

        width: 96%;

    }

grid

display에 grid를 주고 areas를 묶어준다.

grid - 반응형

미디어쿼리를 통해 flex 반응형을 만들 수 있다. 


@media (max-width:1220px) {
    .container {
        width: 96%;
    }
    .contents {
        display: grid;
        grid-template-areas: 
            "cont1 cont2 cont2"
            "cont1 cont3 cont4"
        ;
        grid-template-columns: repeat(3, 1fr); /*자동으로 3등분 됨 f:flextion repeat(3,1fr)식으로도 표현가능*/
        grid-template-rows: repeat(2, 1fr)l
    }
    }
    @media (max-width:768px) {
        .container {
            width: 100%;
        }
        .contents {
            grid-template-areas: 
                "cont1 cont2"
                "cont1 cont3"
                "cont1 cont4"
            ;
            grid-template-columns: 30% 70%; /*자동으로 3등분 됨 f:flextion repeat(3,1fr)식으로도 표현가능*/
            grid-template-rows: repeat(3, 1fr);
    }
}
@media (max-width:480px) {
    .contents {
            grid-template-areas: 
                "cont1" 
                "cont2"
                "cont3"
                "cont4"
            ;
            grid-template-columns: 100%; /*자동으로 3등분 됨 f:flextion repeat(3,1fr)식으로도 표현가능*/
            grid-template-rows: 150px 210px 210px 210px;
    }
}
