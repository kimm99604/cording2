01. 선언적 함수

선언적 함수는 함수를 선언할 때 이름을 붙여주는 함수를 말한다. 즉, 따로 변수에 넣어주지 않고 선언할 대 붙여준 이름으로 호출하면 된다.

{
    function func(){
        document.write("함수가 실행되었습니다.1
");
    }
    func(); // 함수를 꼭 호출해줘야 실행됨 
}

02. 익명 함수

변수에 함수 데이터를 저장하여 변수를 마치 함수처럼 사용할 수 있도록 만들어 줍니다.

04. 리턴값 함수

return를 통해 처리된 결과를 반환시켜주는 함수입니다.

{
    function func(){
        const str = "함수가 실행되었습니다.4
";
        return str; //리턴값에 str문자열 저장 /실행문이 있으면 return생략가능 
    }
    document.write(func()); //return값이 있기때문에 document에 함수 선언가능
}

05. 화살표 함수 : 선언적 함수

ECMAscript2016(es6)에 추가된 내용으로 '=>'을 이용하여 함수를 간결하게 표현할 때 사용합니다.

{
    //1.
    func = () => {
        document.write("함수가 실행되었습니다.5
");
    }
    func();

    //2.
    func = () => document.write("함수가 실행되었습니다.5
"); //괄호생략가능 

    func();
}

06. 화살표 함수 : 익명 함수

ECMAscript2016(es6)에 추가된 내용으로 '=>'을 이용하여 함수를 간결하게 표현할 때 사용합니다.

07. 화살표 함수 : 매개변수 함수

ECMAscript2016(es6)에 추가된 내용으로 '=>'을 이용하여 함수를 간결하게 표현할 때 사용합니다.

{
    //1. 
    func = (str) => {
        document.write(str);
    }
    func("함수가 실행되었습니다.7
");

    //2.    
    func = (str) => document.write(str);
    func("함수가 실행되었습니다.7
");

    //3. 앞에 const 생략가능 
    func = str => document.write(str);

    func("함수가 실행되었습니다.7
");
}

08. 화살표 함수 : 리턴값 함수

ECMAscript2016(es6)에 추가된 내용으로 '=>'을 이용하여 함수를 간결하게 표현할 때 사용합니다.

{
    func = () => { //함수안에 두줄 이상은 한줄로 줄일 수 없음 
        const str = "함수가 실행되었습니다.8
";
        return str;
    }
    document.write(func());
}
