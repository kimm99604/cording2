CSS 속성 검색하기

플렉스, 애니메이션, 백그라운드, 기타 등 유형별로 검색해보세요
전체 속성 갯수 : 0
  • accent-color : accent-color 는 특정 요소에 색상을 지정합니다.
  • align-content : align-content 는 콘텐츠 아이템의 상하관계 정렬 상태를 설정합니다.
  • align-items : align-items 는 콘텐츠 아이템의 내부 상하관계 정렬 상태를 설정합니다.
  • align-self : align-self 는 아이템의 교차축 정렬 정하는 속성 입니다.
  • all : all 는 요소의 속성을 초기화 또는 상속을 설정합니다.
  • animation : animation 는 HTML 요소에 적용되는 CSS 스타일을 다른 CSS 스타일로 부드럽게 변화시킵니다.
  • animation-delay : animation-delay 는 적용 대기 시간을지정합니다.
  • animation-direction : animation-direction 는 애니메이션의 진행 방향을 설정합니다.
  • animation-duration : animation-duration 는실행된 애니메이션 진행 시간을 지정하는 속성입니다.
  • animation-fill-mode : animation-fill-mode 는애니메이션 실행 이전이나 이후에 애니메이션 효과를 표시할지의 여부를 지정하는 속성입니다.
  • animation-iteration-count : animation-iteration-count는 애니메이션의 반복 횟수를 설정합니다.
  • animation-name : animation-name 애니메이션 keyframe 이름을 설정합니다.
  • animation-play-state : animation-play-state 애니메이션 진행상태를 설정합니다.
  • animation-timeline : animation-timeline 타이밍 진행률에 따라 출력 값을 생성하는 개체의 Timeline 형식입니다.
  • animation-timing-function : animation-timing-function 특정 애니메이션을 실행할 때 애니메이션의 가속운동, 즉 특정한 시점에서 가속할지 감속할지를 정하는 속성이다.
  • appearance : appearance 요소자체 구성요소를 숨깁니다.
  • aspect-ratio : aspect-ratio 속성은 이미지나 동영상을 비율대로 줄이거나 늘리는 데 사용됩니다.
  • backdrop-filter : backdrop-filter 요소 뒤 영역에 흐림이나 색상 시프트 등 그래픽 효과를 적용할 수 있는 속성입니다.
  • backface-visibility : backface-visibility 속성은 요소의 뒷면이 사용자를 향할 때 보여야 하는지 지정합니다.
  • background : background 백그라운드 속성을 일괄적으로 설정합니다.
  • background-attachment : background-attachment 배경이미지 고정 여부를 설정합니다.
  • background-blend-mode : background-blend-mode 배경을 혼합했을 때, 그래픽 효과를 설정합니다.
  • background-clip : background-clip 백그라운드 이미지의 위치 기준점을 설정합니다.
  • background-color : background-color 백그라운드 색을 설정합니다.
  • background-image : background-image 백그라운드이미지 속성을 설정합니다.
  • background-origin : background-origin 백그라운드 이미지 위치 기준점을 설정하기 위한 속성입니다.
  • background-position : background-position 백그라운드 이미지의 위치 영역을 설정합니다.
  • background-position-x : background-position-x 백그라운드 이미지의 위치 영역 x축을 설정합니다.
  • background-position-y : background-position-y 백그라운드 이미지의 위치 영역 y축을 설정합니다.
  • background-repeat : background-repeat 백그라운드 이미지 반복 여부를 설정합니다.
  • background-size : background-size 백그라운드 이미지 반복 여부를 설정합니다.
  • block-size : block-size 기록 모드에 따라서, 요소의 블록의 수평과 수직 크기를 정의합니다.
  • border : border 테두리 속성을 일괄적으로 설정합니다.
  • border-block-color : border-block-color 속성은 약식 속성 스타일 시트에서 하나의 장소에서 각각의 논리 블록 border 속성 값을 설정합니다.
  • border-block-end : border-block-end 속성은 스타일 시트의 한 곳에서 개별 논리적 블록-엔드 경계선 속성 값을 설정하기 위한 약식 속성입니다.
  • border-block-end-color : border-block-end-color 속성은 요소의 논리적 블록-end 경계선의 색상을 정의하며, 요소의 쓰기 모드, 방향성 및 텍스트 방향에 따라 물리적 경계 색상에 매핑됩니다.
