CSS 속성 갯수 : 0
  • accent-color : accent-color 는 특정 요소에 색상을 지정합니다.
  • align-content : align-content 는 콘텐츠 아이템의 상하관계 정렬 상태를 설정합니다.
  • align-items : align-items 는 콘텐츠 아이템의 내부 상하관계 정렬 상태를 설정합니다.
  • align-self : align-self 는 아이템의 교차축 정렬 정하는 속성 입니다.
  • all : all 는 요소의 속성을 초기화 또는 상속을 설정합니다.
  • animation : animation 는 HTML 요소에 적용되는 CSS 스타일을 다른 CSS 스타일로 부드럽게 변화시킵니다.
  • animation-delay : animation-delay 는 적용 대기 시간을지정합니다.
  • animation-direction : animation-direction 는 애니메이션의 진행 방향을 설정합니다.
  • animation-duration : animation-duration 는실행된 애니메이션 진행 시간을 지정하는 속성입니다.
  • animation-fill-mode : animation-fill-mode 는애니메이션 실행 이전이나 이후에 애니메이션 효과를 표시할지의 여부를 지정하는 속성입니다.
  • animation-iteration-count : animation-iteration-count는 애니메이션의 반복 횟수를 설정합니다.
  • animation-name : animation-name 애니메이션 keyframe 이름을 설정합니다.
  • animation-play-state : animation-play-state 애니메이션 진행상태를 설정합니다.
  • animation-timeline : animation-timeline 타이밍 진행률에 따라 출력 값을 생성하는 개체의 Timeline 형식입니다.
