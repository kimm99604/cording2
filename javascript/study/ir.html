CSS 이미지 스프라이트(Image Sprite)

이미지 스프라이트(image sprite)란 여러 개의 이미지를 하나의 이미지로 합쳐서 관리하는 이미지를 의미합니다.
이미지 스프라이트를 사용하면 이미지를 다운받기 위한 서버 요청을 단 몇 번으로 줄일 수 있습니다.
또한, 많은 이미지 파일을 관리하는 대신 몇 개의 스프라이트 이미지(sprite image) 파일만을 관리하면 되므로 매우 간편합니다.

예제 figma


다음 예제는 하나의 이미지를 가지고 6개의 아이콘을 만드는 예제입니다.
6개의 아이콘을 만들기 위해 6개의 이미지를 사용하는 것이 아닌 다음 이미지 하나만을 가지고 작업하게 됩니다.
간격은 50px씩 주었습니다.

ir효과

ir효과란 이미지에 텍스트 설정하고 안보이게하는 것이다.
웹접근성은 몸이 불편한 사람이라도 정보를 이용하는데 불편함이 없도록 하는것이 목적에 있기 때문에 만약 시각 장애인이 사이트를 이용할 경우 텍스트는 리더기로 읽어줄 수 있지만 이미지는 읽을 수가 없다 그래서 이미지에 텍스트를 붙여준다. 

.ir {  
display: block;
overflow: hidden;
font-size: 0;
line-height: 0;
text-indent: -9999px; /*들여쓰기*/
}

백그라운드(background) 표현

background 속성은 태그의 배경을 지정하는 속성으로, font 속성과 비슷하게 세부적인 속성들을 한번에 쓸 수 있는 속성입니다.
아래의 세부적인 속성을 가집니다.

background-color 배경 색
background-image 배경 이미지
background-repeat 배경 이미지 반복 여부
background-position 배경 위치
