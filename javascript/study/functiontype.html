함수 유형 (Function Type)

함수는 프로그램에서 반복적으로 사용되는 기능을 만들어 내기 위한 코드들의 집합

1. 함수 유형 : 함수와 매개변수를 이용한 형태

코드를 재활용하기 위해 함수를 사용한다. 

function func(num, str1, str2){
    document.write(num+ ". " + str1 + "가" + str2 + "되었습니다. " + "
")
}
func("1","함수", "실행") //1. 함수가 실행되었습니다.
func("2","자바스크립트", "실행") //2. 자바스크립트가 실행되었습니다.
func("3","제이쿼리", "실행") //3. 제이쿼리가 실행되었습니다.

4. 함수 유형 : 객체 안에 변수와 함수를 이용한 형태

5. 함수 유형 : 객체 생성자 함수*****

기존 함수에 new 연산자를 붙여서 호출하면, 해당 함수는 생성자 함수로 동작합니다.
리터럴과 Obejct()로 객체를 생성하는 것과 달리, 생성자 함수를 통해 객체를 생성하면 같은 속성을 가진 객체를 여러 개 생성할 수 있습니다.
생성자 함수로 인스턴스를 생성하기 전에, 먼저 비어있는 객체를 생성합니다. 

function func(num, name, word){ 
    this.num = num; //const를 생략하고, this를 사용함으로써 변수 재활용가능 
    this.name = name;
    this.word = word;

    this.result = function(){
        document.write(this.num + ". "+ this.name +"가" + this.word +"되었습니다. 
");
    }
}

//인스턴트 생성
const info1 = new func("1","함수","실행")
const info2 = new func("2","자바스크립트","실행")
const info3 = new func("3","제이쿼리","실행")

//실행
info1.result(); //1. 함수가 실행되었습니다.
info2.result(); //2. 자바스크립트가 실행되었습니다.
info3.result(); //3. 제이쿼리가 실행되었습니다.

6. 함수 유형 : 프로트타입 함수

함수 프로토타입 은 EGL 시스템 코드가 함수 자체에 액세스하지 못하는 경우 함수 호출에 대해 유형 검사를 수행하는 데 사용되는 정의입니다.
함수 프로토타입은 function 키워드로 시작한 다음 함수 이름, 해당 매개변수(있는 경우), 리턴값(있는 경우)을 나열합니다. 

function func(num, name, word){ 
    this.num = num; //const를 생략하고, this를 사용함으로써 변수 재활용가능 
    this.name = name;
    this.word = word;
}

//result를 함수 밖으로 빼줌 
func.prototype.result = function(){
document.write(this.num + ". "+ this.name +"가" + this.word +"되었습니다. 
");
}

//인스턴트 생성
const info1 = new func("1","함수","실행")
const info2 = new func("2","자바스크립트","실행")
const info3 = new func("3","제이쿼리","실행")

//실행
info1.result(); //1. 함수가 실행되었습니다.
info2.result(); //2. 자바스크립트가 실행되었습니다.
info3.result(); //3. 제이쿼리가 실행되었습니다.

7. 함수 유형 : 객체 리터럴 함수

자바스크립트 함수는 식별자 없이 함수 그 자체를 객체 타입의 값으로 리터럴로 생성이 가능합니다.
객체 타입의 값으로 생성된 함수 리터럴의 레퍼런스를 변수에 대입해줌으로써 , 변수명(인자목록) 의 방법으로도 호출이 가능하고, 한번만 호출하고 다시 호출할 수 없는 즉시 실행 함수 (함수 리터럴)으로도 호출할 수 있습니다. 

 function func(num, name, word){ 
    this.num = num; //const를 생략하고, this를 사용함으로써 변수 재활용가능 
    this.name = name;
    this.word = word;
}

//객체 리터럴 함수 
func.prototype = {
result1 : function(){
document.write(this.num + ". "+ this.name +"가" + this.word +"되었습니다. 
");
},
result2 : function(){
document.write(this.num + ". "+ this.name +"가" + this.word +"되었습니다. 
");
},
result3 : function(){
document.write(this.num + ". "+ this.name +"가" + this.word +"되었습니다. 
");
}
}

//인스턴트 생성
const info1 = new func("1","함수","실행")
const info2 = new func("2","자바스크립트","실행")
const info3 = new func("3","제이쿼리","실행")

//실행
info1.result1();  //1. 함수가 실행되었습니다.
info2.result2(); //2. 자바스크립트가 실행되었습니다.
info3.result3(); //3. 제이쿼리가 실행되었습니다.
